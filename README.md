# Step Pyramid of Djoser

An Ancient Egyptian themed math project website featuring a scaled Step Pyramid of Djoser, with measurements, visuals, and QR code access.

## 📐 About the Project
This project demonstrates mathematical scaling using a model of the Step Pyramid of Djoser.

## 🏺 Features
- Scaled pyramid design
- Ancient Egyptian aesthetic
- Measurement display
- QR code access to the website
